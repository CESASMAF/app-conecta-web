Status pill for short labels — system status, counts, categories. Tinted by default; `solid` for high-emphasis.

```jsx
<M3Badge variant="success" dot>Publicado</M3Badge>
<M3Badge variant="warning" icon="schedule">429 — aguarde</M3Badge>
<M3Badge variant="primary" solid>3</M3Badge>
```

Variants: `neutral · primary · success · warning · danger · info`. Pair color with text/icon — never color alone.
