Risk/vulnerability pill from computed analytics. Presets carry color + icon + label.

```jsx
<M3RiskChip risk="overcrowding" />  {/* Sobrelotação */}
<M3RiskChip risk="violation" />     {/* Violação de direitos */}
<M3RiskChip risk="default" label="Insegurança alimentar" icon="restaurant" />
```
Presets: `violation · overcrowding · dropout · prenatal · default`.
